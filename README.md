## 🚀 Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/noshu12/my-portfolio.git
   cd my-portfolio
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a live server for better experience
   ```

### Using Live Server (Optional)
```bash
# Install live-server globally (one time)
npm install -g live-server

# Run in project directory
live-server
```
